# Open project and install dependencies

```bash
npm i
```

## Get access token and add to environment

You may use postman test collection and test environment [here](https://www.postman.com/flight-pilot-5671206/test/collection/iv1oyks/fintacharts-api-v1?action=share&source=copy-link&creator=25580789)
Copy access token from get token post request and paste to "access_token" in environment.development.ts

## Run app

```bash
ng serve
```
